So let’s look up the exif_imagetype function:

exif_imagetype() reads the first bytes of an image and checks its signature.

So it seems that exif_imagetype() checks the magic number of the file we want to upload. Could it be as easy as adding the magic number to our webshell? Let’s try that

sed -i '1s/^/\xFF\xD8\xFF\xE0\n/' webshell.php

Yeah, that worked!
